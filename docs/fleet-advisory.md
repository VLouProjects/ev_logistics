# Fleet Advisory & Cost Evaluation Tool

## Purpose

This tool helps customers understand key cost drivers when evaluating electric logistics vehicles.

It is not a guaranteed ROI calculator.

Because the company supplies multiple brands, vehicle classes, and refrigerated or non-refrigerated configurations, the tool provides indicative planning logic only.

The goal is to support consultation, not replace a tailored assessment.

## Page Name

車隊規劃及成本評估

## Required Inputs

1. Vehicle Class

Options:

- 電動客貨車
- 輕型電動貨車
- 中型電動貨車
- 重型電動貨車

2. Refrigeration Requirement

Options:

- 無需冷鏈
- 冷藏
- 冷凍
- 醫藥溫控
- 尚未確定

3. Daily Distance

Unit:
km/day

4. Fleet Size

Unit:
vehicles

5. Diesel Cost

Unit:
per liter

6. Electricity Cost

Unit:
per kWh

## Output Language

Use:

- 估算柴油每月燃料成本
- 估算電動車每月能源成本
- 估算每月成本差異
- 估算年度成本差異

Do not use:

- guaranteed ROI
- guaranteed savings
- official fuel economy claim
- official range claim

## Required Disclaimer

本工具僅提供初步規劃及成本評估參考，並不代表最終報價、保證節省或實際營運結果。實際成本取決於品牌、車型、載重、路線、充電成本、冷鏈設備負載、維修條件及實際營運情況。
