# Cold Chain Configuration Matrix

## Purpose

This module helps customers understand which refrigeration configuration may fit their industry, cargo type, and vehicle class.

It supports multi-brand electric vans, electric trucks, and refrigerated vehicle options.

It must not imply that the company manufactures vehicles, refrigeration systems, or EV chassis.

## Positioning

This module demonstrates specialist cold-chain advisory capability.

The company helps customers evaluate:

- temperature requirements
- cargo type
- vehicle size
- body configuration
- refrigeration compatibility
- operating impact

## Matrix Logic

The matrix maps:

- Temperature class
- Industry application
- Vehicle class compatibility
- Refrigeration body option

## Temperature Classes

1. 冷藏配送
   - 0°C 至 5°C
   - 蔬果、乳製品、飲品、冷藏餐食

2. 冷凍運輸
   - -18°C 至 -25°C
   - 海鮮、急凍食品、冷凍肉類

3. 醫藥溫控
   - 2°C 至 8°C
   - 醫療用品、保健產品、溫控藥品

4. 多溫層配送
   - 多溫區配置
   - 超市配送、混合食品配送、多品類路線

## Industry Columns

- 餐飲及食品
- 海鮮及冷凍食品
- 醫藥運輸
- 零售配送
- 電商及短途配送

## Compatibility Labels

- 高度適合
- 可考慮
- 需進一步評估

## Vehicle Class Compatibility

Each selected configuration should show compatibility with:

- 電動客貨車
- 輕型電動貨車
- 中型電動貨車
- 重型電動貨車

Use labels:

- 建議
- 可考慮
- 不太適合

Do not guarantee compatibility without brand-specific confirmation.

## Detail Panel Requirements

When a matrix cell is selected, show:

1. Selected configuration
2. Operational use case
3. Temperature requirement
4. Vehicle class compatibility
5. EV operating impact
6. Advisory disclaimer

Required disclaimer:

以上內容僅作初步方案評估，最終車型及冷鏈配置需根據實際品牌、車款、車廂規格、載重、路線及營運條件確認。
